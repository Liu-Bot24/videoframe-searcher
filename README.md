# VideoFrame Searcher

Languages: [简体中文](README.md) · [English](README-en.md)

</div>

VideoFrame Searcher 是一个桌面工具，用于从在线视频或本地视频中快速提取截图，并把抽帧、管理与后续以图搜图串成一条顺手的工作流。当前支持 Windows 和 macOS。

## 系统要求

- 操作系统：Windows 10/11、macOS
- Python：3.11+
- 浏览器：Chrome（仅“以图搜图”功能需要）

## 安装与启动

### 一键启动（推荐）

首次启动脚本会自动创建虚拟环境、安装依赖并启动程序。

- Windows：双击根目录 `start.bat`
- macOS：双击根目录 `start.command`

如果 macOS 提示没有执行权限，可先运行：

```bash
chmod +x start.command
```

### 命令行启动

```bash
python run.py
```

## 主要功能

- 链接解析：基于 `yt-dlp` 获取标题、时长、封面、来源等元数据
- 视频下载：支持画质优先、代理、Cookie、额外参数与多策略回退
- 本地导入：将本地视频复制到项目目录并直接进入抽帧流程
- 自动抽帧：按固定间隔抽帧，可选清理该视频历史截图
- 手动抽帧：支持多时间点打点，结果可与间隔抽帧合并输出
- 视频播放：拖拽进度、左右键逐帧、当前位置截图、打点批量截图
- 截图画廊：分页浏览、批量选择、删除截图、右键删除、打开截图目录
- 插件联动：选中截图后提交 Chrome 插件执行以图搜图
- 项目管理：历史项目列表、缩略图预览、标题滚动显示、项目删除

## 浏览器插件（以图搜图）

1. 打开 `chrome://extensions/`
2. 开启“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选择仓库目录 `chrome_extension/`
5. 启动桌面程序
6. 在插件弹窗中将状态切换为“开启”

桥接地址：`http://127.0.0.1:38999`

说明：macOS 下“以图搜图”会优先拉起 Google Chrome；如果你使用的是其他 Chromium 浏览器，请先确认扩展也已加载到对应浏览器。

## 使用流程

1. 在“采集工作台”输入视频 URL，点击“解析元数据”
2. 选择画质优先后点击“开始下载”，或直接上传本地视频
3. 设置抽帧间隔与手动时间点，执行“开始抽帧”
4. 在“视频播放”中定位帧并执行截图当前/截图打点/批量截图
5. 在“截图画廊”中筛选、删除或执行“以图搜图”

## 配置项

设置页支持以下配置：

- 工作区目录
- Cookie 授权（浏览器或 cookies.txt）
- HTTP 代理
- 下载格式表达式
- 合并输出格式
- yt-dlp 额外参数
- yt-dlp 强制更新

## 常见问题

### 无法启动

优先检查：

- `logs/launcher.log`
- `logs/app.log`

### 无法解析或下载

检查以下项目：

- 链接可访问性
- 登录态与 Cookie 是否有效
- 代理设置是否可用
- yt-dlp 是否为最新版本

### 以图搜图无响应

- 确认 Chrome 插件已安装并开启
- 确认桥接服务在线
